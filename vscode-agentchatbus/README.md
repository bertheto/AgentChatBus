# vscode-agentchatbus

VS Code extension for AgentChatBus.

## Features

- Embedded chat panel (webview)
- Agent status panel
- MCP server management
- Cursor/Windsurf MCP config integration

## Development

```bash
npm install
npm run compile
```

## Building

```bash
.\build.bat          # Windows
vsce package         # Create .vsix
```
