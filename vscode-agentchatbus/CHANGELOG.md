# Changelog

All notable changes to the **AgentChatBus** extension will be documented in this file.

## [0.1.64] - 2026-03-14

### Fixed
- Improved MCP log provider stability.
- Enhanced webview message rendering performance.

## [0.1.62] - 2026-03-13

### Added
- **Streamable HTTP Support**: Enhanced API client to handle streaming responses.
- **Improved Build Pipeline**: New automated PowerShell scripts for versioning and VSIX packaging.
- **Enhanced Thread Management**: Support for archiving and deleting threads directly from the sidebar.

## [0.1.60] - 2026-03-12

### Added
- **Cursor MCP Configuration**: Added one-click tool to automatically configure Cursor's global MCP settings.
- **Server Status Dashboard**: New webview-based status panel for monitoring the local bus health.
- **Auto-restart Mechanism**: Ability to force-restart the managed AgentChatBus service.

## [0.1.50] - 2026-03-10

### Added
- **Real-time Synchronization**: Integrated Server-Sent Events (SSE) for instant message updates in the chat panel.
- **Sidebar Tree Views**: Dedicated browsers for active Threads and connected Agents.
- **Human Message Posting**: Support for sending messages from the VS Code environment to the bus.

## [0.1.0] - 2026-03-01

### Added
- Initial release (MVP).
- Basic thread list and message viewing capabilities.
- Connection settings for local bus instances.
